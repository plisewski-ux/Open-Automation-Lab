# PB-001 - Build Reference Platform

## Purpose

Build a complete Open Automation Lab platform from a clean Ubuntu installation.

## Workflow

1. Provision VPS
2. Configure SSH
3. Create Administrator Account
4. Install Docker Engine
5. Create Platform Filesystem
6. Deploy Reverse Proxy
7. Deploy Core Services
8. Configure Backup
9. Configure Monitoring
10. Perform SSH Hardening
