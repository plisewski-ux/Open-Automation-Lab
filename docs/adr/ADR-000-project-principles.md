# ADR-000 - Project Principles

## Status

Accepted

## Cel

Projekt Open Automation Lab ma być długoterminowym środowiskiem R&D.

## Zasady

- Infrastructure as Code
- Docker jako domyślny sposób wdrażania
- Dokumentacja powstaje równolegle z implementacją
- Wszystkie decyzje architektoniczne są dokumentowane
- Bezpieczeństwo przed wygodą
- Najpierw zrozumienie, potem implementacja
- Każdy etap kończy się działającym rezultatem
